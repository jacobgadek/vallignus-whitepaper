## About Vallignus

Vallignus proposes runtime governance as a distinct control primitive for autonomous systems.

Rather than shaping model behavior through alignment or output filtering, runtime governance enforces policy directly at execution time, ensuring unsafe, excessive, or unauthorized actions cannot occur.

This approach is intended for environments where autonomy must coexist with enforceable boundaries, including enterprise infrastructure and government systems.
